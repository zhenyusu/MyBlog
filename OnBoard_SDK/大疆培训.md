## 编译运行OSDK和例程

1. 获取源代码
   `git clone https://github.com/dji-sdk/Onboard-SDK.git`
2. cmake：自动设置源文件，头文件和库函数路径，生成编译配置文件makefile
* `cd Onboard_SDK && mkdir build && cd build && cmake ..`
3. make: 执行编译和链接
4. 编写UserConfig.txt
5. 运行
* 可执行文件在bin目录下
*  `bin/djiosdk-telemetry-sample ~/UserConfig.txt`
