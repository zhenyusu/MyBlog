## 进程控制

### fork()

> **功能**: 创建一个子进程。**子进程会接着父进程的fork()语句后继续执行。**
> **返回值**: 失败返回-1；  成功返回：(1)父进程返回子进程的ID(非负) (2)进程返回 0 
> **注意**: pid_t类型表示进程ID，但为了表示-1，它是有符号整型。(0不是有效进程ID，init最小为1)。注意返回值，不是fork函数能返回两个值，而是fork后，fork函数变为两个，父子需各自返回一个。


**Demo**:循环创建n个进程
错误写法：
```cpp
for(i = 0; i < n; i++)
    fork();
```
